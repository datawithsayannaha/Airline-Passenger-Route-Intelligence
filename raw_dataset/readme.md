> **Note:** The raw airline datasets were imported and transformed in PostgreSQL to build a dimensional data warehouse using fact and dimension tables.
The processed warehouse served as the primary data source for all Tableau dashboards, enabling airline operations analysis, route intelligence, passenger analytics,
revenue performance monitoring, and delay impact assessment.
